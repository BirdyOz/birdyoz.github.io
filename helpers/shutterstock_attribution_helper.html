Shutterstock attribution maker

A Bookmarklet to create HTML snippets to create Shutterstock attributions and properly align these statements with the image. These can be pasted into any HTML editor.

First time user?

  1. Drag the following bookmarklet into your browers bookmark bar: Shutterstock attribution maker
    (Note: Make sure you drag the link onto your browser bookmarks. Clicking will just bring you back to here)
  2. Visit Shutterstock and locate the image you wish to attribute
  3. Click on the 'Shutterstock attribution maker' in your bookmarks
  4. You will be returned to this page, but now your selected image will be displayed below
  5. Press the copy button next to the layout type that you prefer. This will copy the HTML to your clipboard
  6. Paste this HTML anywhere that you can paste code (eg a Moodle page or tool)
