Turning your Word documents into web compatible content

If you want to convert your Microsoft Word documents to an HTML format you can simply save them as a Web Page by selecting Save as and then saving as Web Page (*.htm,*.html) option. Your Word document creates a web page and a folder, containing any images that were include in the Word document. The problem with this method of creating web pages is that the html code that is created in the converted file contains a lot of proprietary code which makes the files large. It may also prevent the web pages from working correctly in a learning management system.

There are also programs that will convert Microsoft Word documents to web pages for you. Two of these are described here.

Wimba Create

Wimba Create is a product that converts a Word document to a series of web pages in a very clean manner. It is designed specifically for creating educational resources from Word documents. You can also add interactive elements such as quiz questions, flash cards and media directly into the word document. The Word document can be saved as a web site, a single web page or a SCORM learning object for uploading into your learning management system. This is not a free product but you can access a free trial version.

View how Course Genie works http://www.wimba.com/demos/wimbacreate.php

Word to HTML Converter

You can also convert Word to HTML using Word to HTML Converter 1.5.5 This is an application that simply turns Word into web pages and saves any images in a separate folder. You can upload the converted file and the folder to your web site or into your learning management system. This product is not specifically designed for the development of educational resources. This is a Shareware product which means you can download a free trial version but you will need to buy the product when the trial period has expired. It is quite cheap.
