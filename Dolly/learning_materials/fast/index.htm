Making learning materials (fast)

Easy-to-use tools for non-specialists

As well as saving time and money by finding, assembling, customising or generating learning materials, there is a growing  trend towards using tools which enable the rapid development of new e-learning content.

These  tools can be used by teachers and trainers willing to spend the time learning the necessary skills and so have control over preparing their own materials, or they can be used by specialist developers and teams to help reduce the time and cost of producing content for e-learning.
