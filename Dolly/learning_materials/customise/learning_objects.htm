Customising learning objects

The Australian Flexible Learning Framework's Toolbox Learning Object Repository has a series of multimedia based tutorials that explain how to customise learning objects. These tutorials cover the following topics:

  • Using Toolbox learning objects with personal portals.
  • Uploading and using videos in learning objects.
  • Uploading and using images in learning objects.
  • Making simple changes to images in learning objects.
  • Creating and adding audio files to learning objects.
  • Linking to blogs and wikis in learning objects.
  • Making simple text edits to learning objects.

There are several things you need to be aware of when you customize a learning object in the form of a SCORM package.

A SCORM package contains a special file called an IMS Manifest file. This file lists all the files in the package and how the navigation works between the files. If you add extra web pages (or html files) to the SCORM package the IMS Manifest will not include these web pages in its structure. If you want to add more files then you will need to repackage the SCORM package using a program such as Reload editor. This will allow you to easily recreate the IMS Manifest file. This is demonstrated in the section Assemble it.

The next section on Customising web pages tells you about techniques you can use to add, alter and delete in the web pages in the SCORM package. To do this you first need to unzip the package using a program like WinZip. When you have finished your customization zip the package again. Remember do not add or delete any web pages as the SCORM package will not work when loaded into your learning management system.
