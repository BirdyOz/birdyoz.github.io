Using indexes

Using indexes to locate learning resources

There are a large number of sites on the internet that have collected together the links for educational web sites. These indexes give you details about the resource and have a link that you use to visit the resource site. In most cases you will not be able to download these resources. To use the resource you will need to create a link for your students to follow to the site.

The web has many resources that can be linked to support your teaching and learning needs. The resources are web pages that are located on other organisations’ servers.
There are a number of sites that have created indexes of learning resources and list them with a description and a link to the resource.

Most of the indexes are organized according to subject area or fields of study and have search facilities to help you locate the resource you are looking for.

You will not be able to download these resources. To use the resources listed in the index you need to make a hyperlink to the resource’s home page.

Resources

Web Resources

  • Merlot http://www.merlot.org/merlot/index.htm
    Merlot is a leading edge, user-centered, searchable collection of peer reviewed, higher education, online learning materials created by registered members and a set of faculty development support services.
  • World Lecture Hall http://web.austin.utexas.edu/wlh/
    The World Lecture Hall publishes links to pages created by faculty worldwide who are using the Web to deliver course materials in any language.
  • TAFE NSW Learning Gateway – Learning Resources Index http://www.lg.tafensw.edu.au/index/indexlr.htm
    This index provides links to Australian industry web sites arranged by program and curriculum areas.
  • ABC Education Resources – Learn Online  http://www.abc.net.au/learn/
    The ABC provides a rich collection of web resources to complement many of its radio and television programs. Transcripts and podcasts of many programs can also be accessed from the site. The resources can be searched by subject area as well as educational level.
  • BBC - Schools – Homepage http://www.bbc.co.uk/schools/
    The BBC Schools Homepage provides a large range of very engaging learning resources. These are primarily focused at primary and secondary school levels but there may be resources that are also appropriate or some levels of VET studies.

Images and other media

Images and media, that are free to use, can be found at the sites below:

  • Flickr    http://www.flickr.com/creativecommons/
    Flickr has millions of photographic images available for you to select from. Many Flickr users have chosen to offer their work under a Creative Commons license. This allows you to browse or search through photos under each type of license.
  • Creative Commons http://creativecommons.org/find/
    Creative commons licensed resources can be accessed and used for free. A Creative Commons Copyright licence specifies how you can use the resources. This can be for free use so long as you attribute the original creator, or you may be able to make derivatives from the work or use it in non-commercial activities.
  • Teacher Tube  http://www.teachertube.com/
    Teacher Tube provides an online community for sharing instructional videos. Video resources can be accessed or uploaded. The site also provides scripts that allow you to embed the video in your own web pages.

Wikis

  • WikiEducator  http://www.wikieducator.org/Main_Page
  • Wikispaces http://www.wikispaces.com
    Wikispaces contains over 16,000 entries that have been created by students and teachers. Many of these are open for public contributions and editing of information. If you find content that is useful you can create a link to the wikispace page.
