Convert PowerPoint to web pages

If you have some Microsoft PowerPoint presentations you would like to include in your e-learning resources you can convert PowerPoint slides to web pages. You can save the PowerPoint file as a web page. In the “Save as” option select the Web Page (*.htm, *html) file type. This converts each slide into a web page and images. You will lose any animations in your original PowerPoint file. The converted PowerPoint presentation is shown in a web page with a table of contents based on the slide titles. It is important to add meaningful titles to each slide before you convert the file.

There are a few issues with this method of converting PowerPoint files. There is no extra information other than what is on the slides. The animations are also lost from the presentation.

Converting PowerPoint to Flash

The following methods, which convert the PowerPoint files to a Flash file, may provide a better solution. Flash files are able to maintain PowerPoint Animations as well as let you add voice narration. As Flash is embedded in a web page you have the opportunity to edit the web page to add other information to support what is being viewed in the Flash based presentation.

FlashPoint

FlashPoint converts PowerPoint files into Flash files. These are usually smaller files and allow you to add narration to each slide. When you install FlashPoint it is added into your PowerPoint application as an extra menu item. When you convert your presentation you are given options as to the layout of the presentation, whether it is to loop or not and what timing to include on the automatic progression of the slides. You can also select a background audio file to be played with the presentation or record a narration for each slide.

The output from the conversion using Flashpoint is a single web page in which the flash file is embedded. You can edit this web page to provide the context for which the presentation is being shown.

FlashPoint is a shareware product and you will need to purchase it after the trial period has expired.

View an example of a PowerPoint presentation converted to Flash using FlashPOINT at: http://www.flashdemo.net/powerpoint-to-flash/gallery.html

AuthorPOINT Lite

AuthorPOINT lite is a stand-alone application which converts PowerPoint files into a web based flash file. If you have added animations or narrations to your original PowerPoint presentation these are retained in the Flash presentation file. Animations and narrations will, however, create larger files that will take longer to download when being viewed online.

Once converted to Flash, presentations can be viewed in any web browser, regardless of the operating system.

This product is free to download and use

View an example of a PowerPoint presentation converted to Flash using AuthorPOINT at: http://www.authorgen.com/authorpoint-lite-free/powerpoint-presentation-examples.aspx
