Preserve Aspect Ratio

You can force to preserve an aspect ratio. Use minSize not only to define crop area's minimum width and height, but crop aspect ratio too.


$('#image-4').rcrop({
    minSize : [160,90],
    preserveAspectRatio : true,
    grid : true
});
