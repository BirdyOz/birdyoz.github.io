Finding learning materials

Locating and accessing existing materials

The internet is a rich source of content which can be used to support your e-learning solution. It's useful to separate these materials into two main categories:

  • Courseware
    That is, "software for courses" — purpose-built, computer-based resources which have been designed for education and training settings. They typically follow quality standards for layout and learning design, and feature various forms of computer-level interactivity with learners.
    You can usually customise courseware for you needs.
  • Web resources
    Any information available on the web which is relevant to the learning aims.
    You normally need to use web resources unchanged, simply by linking to them, unless you have arrangements with the owner/author/publisher.
